How To Use Bubble/Dew/Flash Calculator

1. Select the components that you want to include in the calculation by clicking the drop down arrow.


2. Input the moles or mass for each components. You can also input the fractions instead.


The table from problem example should look like this:


3. Select the calculator you want to use. For the problem example, we use PT Flash.


4. Input system operating condition:
  • For PT Flash calculation, input both temperature and pressure.
  • For Bubble/Dew T calculation, input pressure only.
  • For Bubble/Dew P calculation, input temperature only.
